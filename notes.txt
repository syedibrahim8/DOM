What is DOM?
The dom is a tree-like structure created by the browser from your html.

                          html
                       /       \ 
                      /         \
                     /           \
                    HEAD          body
                  /   |  \         |  \
                meta link title    p  div
                                  /      \
                                strong    img
                                  |
                                  hello

It allows javascript to read, modify, add, delete, changes to HTML & CSS.
HTML = structure
CSS = style
DOM = bridge between HTML & javascript
